Javascript 개요

Script 언어란?
기본 프로그램의 동작을 사용자의 요구에 맞게
수행되도록 해주는 용도로 사용되는 언어.
장점 : 빠르게 배우고, 소스코드 길이가 짧다.
Javascript 란?
웹 브라우저에서 많이 사용하는 인터프리터 방식(동적 로딩)의 객체지향 프로그래밍 언어.
ECMA(웹 기준 협회) 스크립트 표준을 따르는 대표적인 웹 기술이다.

---------------------------------------------------------------------------------------
1) 기본적으로 프로그래밍 언어를 사용하려면 컴파일이 필요함
2진수로 되어있는 코드파일 전체를 메모리에 올려서 사용
인터프리터 방식은 코드의 부분부분을 필요할 때마다 메모리에 올려서 사용
2) 브라우저에서 컴파일 함.

자바스크립트 작성 방법

            소스코드는 함수 단위로 작성하고, 이벤트 속성을 사용하여 구동되게 함. 

            이벤트 속성은 태그마다 차이가 있음. 
                on이벤트명 = "실행할 함수명([전달값])"
