01. Javascript 개요

Javascript 개요

Script 언어란?
기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도로 사용되는 언어
매우 빠르게 배우고, 소스코드 길이가 짧다.
Javascript란?
웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어
ECMA 스크립트 표준을 따르는 대표적인 웹 기술이다.

자바스크립트 작성 방법

        소스코드는 함수 단위로 작성하고, 이벤트 속성을 사용하여 구동되게 한다.

        이벤트 속성은 태그마다 차이가 있다.

        on이벤트명 = "실행할 함수명([전달값])";

inline 방식

태그에 직접 간간한 소스코드를 작성해서 실행되게 하는 방법

마우스오버 / 마우스아웃

internal 방식

script 태그 영역에 작성해서 실행되게 하는 방식
head, body태그에 작성이 가능하다.

external 방식

별도의 js파일로 코드를 작성하여 가져다 사용하는 방법
