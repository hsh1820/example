08. 배열

배열

자바스크립트 배열이란?

자바스크립트에서는 별도의 자료형 지정이 없기 때문에
배열의 각 요소는 모든 자료형을 보관할 수 있는 형태가 된다.
자바의 컬렉션(리스트)와 비슷한 형태가 된다.

배열의 선언

배열 선언 시 배열의 크기를 정하거나, 정하지 않고 선언할 수 있다.

배열 객체의 메소드(함수)

자바스크립트에서는 배열도 object이기 때문에
배열에서 활용할 수 있는 메소드 (함수)가 제공된다.

indexOf() : 배열에서 요소가 위치한 인덱스 리턴

concat(배열명...) : 다수의 배열을 결합

toString(), join() : 배열을 결합하여 하나의 문자열로 반환한다.

toString() : 요소 사이에 ,(콤마)를 추가하면 합친다.
join() : 요소 상에 구분자를 지정하여 합친다.

sort() : 배열을 오름차순으로 정렬
reverse() : 배열의 순서를 뒤집음.

원본 배열의 데이터 순서가 변함

push(): 배열의 맨 뒤에 요소 추가
pop() : 배열의 맨 뒤에 요소 제거

shift(): 배열의 맨 앞에 요소 제거
unshift(): 배열의 맨 앞에 요소 추가

slice() : 배열의 요소 선택 잘라내기
splice() : 배열의 index 위치의 요소를 제거하고 새로운 요소 추가
