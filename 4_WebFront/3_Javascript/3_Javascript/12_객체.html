12.객체

객체

객체 선언 및 속성 호출

자바스크립트 객체는 Key:Value(속성) 형태로 이루어짐.
객체는 {}(중괄호, 객체 리터럴) 또는 new를 이용하여 생성 가능하고,
속성에는 모든 자료형이 올 수 있다.

객체의 메소드 속성

객체의 속성 중 타입이 함수(function)인 속성을 메소드라고 한다.

객체 속성 반복 접근

for in 문 : 객체의 속성에 순차적으로 접근

객체에 사용할 수 있는 키워드 (in, with)

in : 객체 내부에 해당 속성이 있는지 확인하는 키워드
with : 객체 호출 시 객체명을 생략시켜 코드를 감소시키는 키워드





객체의 속성 추가 및 제거

처음 객체 생성 이후 속성을 추가/제거하는 것을 '동적으로 속성을 추가/제거 한다'하고 표현함.
