시각적 효과

            
Effect 메소드

            - 페이지 내에서 애니메이션 효과를 만들기 위한 메소드 집합

            - 이펙트 메소드 작성법
            1) $('s').메소드명();
            2) $('s').메소드명(speed);
            3) $('s').메소드명(speed, easing, callback);

            speed : 애니메이션의 실행 속도(ms 단위) | slow, fast
            easing(완화) : 애니메이션의 지점별 속도 / linear, swing 가능
            callback : 메소드 실행 후 실행할 함수

show() / hide()

            - show() 호출 시 해당 요소는 
              display : none -> 해당 요소의 지정된 display로 변경

토글(toggle)

버튼 하나로 on/off 등 두 가지 동작을 하는 버튼(스위치)
